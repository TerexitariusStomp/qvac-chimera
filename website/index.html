Architecture

How it works

Three layers. One app window.

1

Tauri Desktop Shell

The native window bundles the React frontend and loads it via the internal Tauri protocol. Start/Stop Supervisor controls live in the native header.

2

Supervisor Sidecar

A Go binary bundled as externalBin manages the Docker container lifecycle: pull image, start container, health checks, log streaming.

3

Hardened Container

The Docker container runs Node.js, @qvac/sdk, miners, P2P networking, and the wiki API server on localhost:3002. Non-root user. Minimal deps.
