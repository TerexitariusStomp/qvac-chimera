# qvac/scripts

Helper scripts for the QVAC node.

## Files

- `generate-cortensor-node.js` — *(removed)* Cortensor is no longer supported
- `setup-fortytwo-node.sh` — *(removed)* Fortytwo is no longer supported
- `weekly-fund-sweep.js` — Weekly fund sweep into the protocol EVM collection multisig

## Notes

The QVAC node delegates most tasking-network setup to the root-level `providers/` and `scripts/` directories. The remaining files here are QVAC-specific operational helpers.
