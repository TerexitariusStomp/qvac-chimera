# qvac/backend

Backend language-specific subprojects for QVAC.

## Subdirectories

- `earnidle-docs/` — Earnidle documentation / reference materials
- `earnidle-runtime/` — Earnidle runtime integration
- `rust-project/` — Rust components (e.g., Tauri sidecar, native extensions)
- `smart-contracts/` — Smart contract source and deployment helpers

See also `../src/` for the primary Node.js backend.
