# qvac/app

Python application layer for QVAC.

## Files

- `flask_main.py` — Flask server entry point
- `main.py` — Main application logic
- `models.py` — Data models
- `schemas.py` — Validation schemas
- `semantic_engine.py` — Semantic search / similarity engine

## Purpose

This directory contains the legacy Python QVAC app components. New development is primarily in `../src/` (Node.js backend).
