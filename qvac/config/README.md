# qvac/config

QVAC configuration files.

## Files

- `config.json` — Live QVAC configuration
- `config.example.json` — Example/template configuration

## Included Configs

- Local LLM settings
- Tasking network miner settings (Chutes, Routstr, BTT AI, Golem, Anyone, Mysterium, BTFS, Casper, Earnidle)
- Casper relay settings
- Wallet / payout configuration

Copy `config.example.json` to `config.json` and customize for your deployment.
