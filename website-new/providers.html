Casper Escrow Markets

Chimera nodes register on all four Casper market contracts for on-chain job assignment and escrow-settled payments.

InferenceMarket
LLM inference via WebGPU/WebLLM. Browser nodes and desktop apps serve inference jobs.
StorageMarket
IPFS/Helia file storage. Browser nodes pin and serve files via IPFS.
ComputeMarket
WASI/Wasmer sandboxed compute. Browser nodes run WASM workloads.
BandwidthMarket
WebRTC bandwidth sessions. Browser nodes provide relay bandwidth.
